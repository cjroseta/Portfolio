# Relatórios Executivos

Relatórios orientados à gestão, construídos a partir de dados operacionais centralizados. A prioridade é transformar dados de campanhas, contratos, ocupação e performance em informação útil para decisões.

O desenho deve manter métricas rastreáveis à fonte, critérios de cálculo documentados e uma separação clara entre dados operacionais e apresentação executiva.
