# Relatórios de Veiculação de Campanhas

Automação de relatórios a partir de dados operacionais de campanhas e painéis. O objectivo é reduzir a dependência de produção manual e disponibilizar informação consistente sobre execução, ocupação e entrega.

Os relatórios usam a integração entre Scala Content Manager, Odoo e dados de BI como base para validação e comunicação com o negócio.
