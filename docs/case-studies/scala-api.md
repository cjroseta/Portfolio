# Scala Content Manager API

## Arquitectura

Scala Content Manager → REST API → Odoo → Business Intelligence.

## Capacidades

- Sincronização de campanhas e ocupação de painéis.
- Gestão do ciclo de vida de campanhas.
- Dados para relatórios de veiculação, ocupação e comparação contratado versus entregue.
- Centralização de informação operacional no Odoo e suporte a dashboards executivos.
